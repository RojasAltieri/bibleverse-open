## 0.1.0

- Initial open-source release of `bibleverse_open`.
- Added reusable client-side widgets:
  - `VerseReaderCard`
  - `StreakOverviewCard`
  - `ChallengeCard`
  - `ProgressBadgeChip`
  - `ProgressBadgeWrap`
- Added `BibleVerseOpenTheme.light()` for quick app theming.
- Added safe mock models and sample data through `BibleVerseMockData`.
- Added runnable `example/` showcase app for mobile/web/desktop testing.
- Added `CONTRIBUTING.md` with strict client-side contribution policy.
- Added MIT license.
